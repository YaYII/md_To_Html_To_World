Markdown 轉 Word 插件

一鍵將 Markdown 文檔轉換爲格式精美的 Word 文檔（.docx 格式）。 有什麼問題，可以直接抖音小紅書私聊。

作者：楊英 抖音號：YI.11.13 小紅書：YI_11_13

創作不易，給個關注。愛護你我他。生態靠大家。沒錢恰飯了，只能用精神食糧麻醉自己了。

已婚男士，私聊只談技術，不談其他（我老婆讓我加的 Orz）。

功能特點

  • 🚀 一鍵轉換：右鍵點擊 Markdown 文件即可轉換爲 Word 文檔
  • 🎨 精美排版：支持自定義字體、顏色、段落樣式等
  • 📊 表格支持：完美呈現 Markdown 表格，並應用表格樣式
  • 💻 代碼高亮：保留代碼塊格式和語法高亮
  • 📷 圖片處理：自動將圖片嵌入到 Word 文檔中
  • 🔄 批量轉換：支持轉換整個目錄下的 Markdown 文件（命令行方式）

使用方法

基本用法

  1. 在 VS Code 中打開一個 Markdown (.md) 文件
  2. 右鍵點擊文件內容或在文件資源管理器中右鍵點擊文件
  3. 選擇 "轉換爲 Word 文檔"
  4. 轉換完成後，會提示轉換成功，並提供打開文件的選項

設置 Python 解釋器路徑

默認情況下，插件會自動檢測系統中的 Python 環境。如果自動檢測失敗或想使用特定的 Python 環境，可以在設置中指定：

  1. 打開 VS Code 設置（文件 > 首選項 > 設置）
  2. 搜索 "markdown-to-word.pythonPath"
  3. 輸入 Python 解釋器的完整路徑，例如：
  4. Windows: C:\\Python39\\python.exe
  5. macOS/Linux: /usr/bin/python3

必要依賴

本插件依賴以下 Python 包：

  • python-docx：用於創建和編輯 Word 文檔
  • markdown：Markdown 解析器
  • beautifulsoup4：HTML 解析
  • opencc-python-reimplemented：簡繁中文轉換
  • pyyaml：配置文件解析
  • lxml：XML 和 HTML 處理

插件會在首次運行時檢測並自動安裝所需依賴，如果自動安裝失敗，請手動安裝：

pip install python-docx markdown beautifulsoup4 opencc-python-reimplemented pyyaml lxml

高級配置

插件會使用合理的默認配置，但您可以通過編輯配置文件來自定義轉換效果。具體請參考插件目錄中的 scripts/config_example.yaml 文件。

常見問題

  1. Q: 轉換後的文檔中文字亂碼怎麼辦？
    A: 請確保您的系統安裝了插件配置中指定的字體，或修改配置文件中的字體設置。

  2. Q: 依賴包安裝失敗怎麼辦？
    A: 請嘗試手動安裝所需依賴，或檢查網絡連接和 Python 權限設置。

  3. Q: 如何批量轉換整個文件夾的 Markdown 文件？
    A: 目前批量轉換功能僅在命令行模式下可用，請在終端中運行插件目錄下的 run.py 腳本。
